# DESIGN_SYSTEM

## 用途

记录网站视觉与交互规则，确保前台和后台界面风格统一，并同时适配电脑端和手机端。

## 设计目标

- 高级、通透、轻盈、现代、清爽、有层次。
- 参考 iOS 风格的液态玻璃视觉，但不要过度花哨。
- 更像高质量中文工具发现站，不做夸张营销页。
- 用户能快速浏览、筛选、判断一个工具是否值得继续阅读详情。
- 后台界面以简单、直观、少干扰为主。

## 整体风格

- 使用浅色高级风格，页面整体保持明亮、干净、可信。
- 大量留白，避免信息堆叠和拥挤。
- 使用柔和渐变背景区分不同页面分区，不能全页面一片白。
- 重要内容使用半透明玻璃卡片，形成通透感和层次感。
- 使用柔和阴影和细边框，不使用厚重、强烈、廉价的阴影。
- 所有页面保持统一视觉语言，包括首页、工具列表页、工具详情页、文章页和后台页。

## 颜色规范

### 主背景

- `#f8fbff`
- `#f5f7fb`
- `#eef6ff`

### 主文字

- `#0f172a`

### 次级文字

- `#475569`
- `#64748b`

### 强调色

- `#0ea5e9`
- `#14b8a6`
- `#6366f1`

### 柔和渐变

- 蓝白渐变。
- 青蓝渐变。
- 紫蓝渐变。
- 浅灰白渐变。

## 液态玻璃卡片规范

所有重要卡片优先使用统一玻璃态视觉：

```css
background: rgba(255, 255, 255, 0.68);
backdrop-filter: blur(20px);
border: 1px solid rgba(255, 255, 255, 0.65);
box-shadow: 0 24px 80px rgba(15, 23, 42, 0.08);
border-radius: 24px;
```

使用规则：

- 适用于 Hero 概览卡、搜索面板、工具卡片、推荐内容卡片、广告位和重要信息面板。
- 卡片内部必须有足够内边距，避免文字贴边。
- 卡片之间保持舒适间距，避免形成表格感。
- Hover 状态可以轻微上浮，并增强阴影，但不要过度动画。

## 分区规范

首页和工具页必须分成明显区域：

1. Hero 区
2. 搜索区
3. 分类区
4. 推荐内容区
5. 最新内容区
6. 广告区
7. 页脚区

分区设计规则：

- 每个分区使用不同但协调的浅色渐变背景。
- 分区之间要有自然过渡，避免硬切。
- 分区标题需要清晰，但不要使用过度装饰。
- 重要内容区可以使用玻璃卡片承载，普通内容区保持轻量。
- 广告区必须明确但低调，不影响正文阅读。

## 按钮规范

### 主按钮

- 深色背景。
- 白色文字。
- 圆角 `14px`。
- 按钮高度在移动端不低于 `44px`。

### 次按钮

- 半透明白色背景。
- 浅色边框。
- 深色文字。
- 适合用于筛选、返回、辅助入口。

### Hover 状态

- 轻微上浮。
- 阴影增强。
- 背景颜色有轻微变化。
- 动画时间保持短而自然。

## 广告位规范

广告位必须是明确但不突兀的模块。

广告位可使用文案：

- 广告位
- 合作推广
- 此处可展示赞助工具或推广内容

广告位样式：

- 浅色玻璃卡片。
- 虚线边框。
- 低饱和文字。
- 留白充分。
- 不使用强烈动效。
- 不影响正文阅读。

## 响应式规则

- 所有页面必须兼容手机端和电脑端。
- 手机端优先保证阅读和点击舒适。
- 电脑端优先保证信息浏览效率。
- 不使用固定宽度导致横向滚动。
- 所有卡片在移动端默认单列。
- Hero 标题在移动端不能超过 `36px`。
- 移动端分区间距可以缩小，但必须保持呼吸感。
- 移动端按钮高度不低于 `44px`。

## 颜色原则

- 使用少量主色，渐变要柔和、克制、统一。
- 背景保持干净，正文对比度要足够。
- 状态颜色保持一致：
  - 成功：绿色
  - 警告：黄色或橙色
  - 错误：红色
  - 信息：蓝色

## 排版原则

- 中文正文优先保证清晰可读。
- 标题层级要明确，不滥用超大字号。
- 卡片、列表、表单的间距保持一致。
- 按钮文字短而明确。
- 正文行高要舒适，避免密集压迫。
- 首页和详情页要通过字号、字重、间距建立清晰层级。

## 组件原则

- 工具卡片展示关键信息：名称、简介、分类、标签、适合人群、状态和详情入口。
- 表单字段命名清楚，必要时给出简短提示。
- 后台操作按钮要区分主操作、普通操作和危险操作。
- 所有可点击区域在手机端要足够大。
- 广告、版权声明、筛选面板等通用模块必须组件化，避免页面内重复写样式和文案。

## 暂定基础断点

- 手机端：默认布局
- 平板端：`md`
- 电脑端：`lg`

具体 Tailwind 配置以后随项目初始化确认。
