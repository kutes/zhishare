# 内容模板

本文件用于统一工具页、文章页、社交媒体推广文案的写作结构。

## 1. 工具详情页内容模板

适用于后台工具表单。

### title

工具名称。

示例：

```txt
Canva
```

### slug

英文小写短横线。

示例：

```txt
canva
```

### summary

一句话结论，5 到 50 字。

写法：

```txt
Canva 更适合不会设计、但需要快速做封面、海报、简历和社交媒体图的新手用户。
```

不要写：

```txt
Canva 是一款功能强大的在线设计工具。
```

### description

结构：

1. 它是什么。
2. 真正解决什么问题。
3. 和小白有什么关系。
4. 不要夸大。

模板：

```txt
Canva 是一个在线设计工具，主要解决的是“不会专业设计软件，也想快速做出能用图片”的问题。它不适合替代专业设计师，但适合用来做封面图、社交媒体配图、简历、海报和简单演示稿。对新手来说，它最大的价值不是功能多，而是模板足够多，能让你先做出一个及格版本。
```

### target_users

适合谁。

模板：

```txt
- 不会 Photoshop，但需要快速做图的人
- 做小红书、抖音、X 等社交媒体内容的人
- 需要做海报、封面、简历、PPT 的学生或个人用户
- 想先用模板完成设计，再慢慢学习排版的人
```

### use_cases

使用场景。

模板：

```txt
- 做社交媒体封面图
- 做课程海报或活动海报
- 做简历、PPT、宣传图
- 快速套模板生成图片内容
```

### pros

优点。

模板：

```txt
- 模板丰富，新手不需要从空白画布开始
- 浏览器里就能用，不需要安装复杂软件
- 中文用户也比较容易上手
- 适合快速出图和批量做内容
```

### cons

缺点，也可以承接“不适合谁”。

模板：

```txt
- 免费版素材和模板有限，部分内容需要付费
- 不适合做复杂专业设计
- 模板用多了容易和别人撞风格
- 商用前需要确认素材和字体授权
```

### risk_notice

风险提醒。

模板：

```txt
如果你要用 Canva 做商业海报、广告素材或客户项目，建议先确认模板、图片、字体和素材的授权范围。免费素材不等于所有场景都能商用，涉及品牌、商单、印刷和公开投放时尤其要谨慎。
```

### website_url

只填写官网或官方产品入口。

模板：

```txt
https://www.canva.com/
```

### download_url

第一阶段必须留空。

规则：

1. 不填写网盘链接。
2. 不填写第三方下载站。
3. 不填写镜像站或安装包直链。
4. 如果工具有客户端，仍然只保留 `website_url` 指向官网入口。

模板：

## 2. 工具详情页完整写作结构

虽然当前后台没有所有字段，但内容策划时统一按照这个结构思考：

```md
# 工具名称

## 一句话结论

这个工具最适合谁，不适合谁。

## 我试过什么

简单说明测试过哪些功能。

## 它真正解决什么问题

不要只解释它是什么，要说它帮助用户解决什么具体问题。

## 哪里容易坑小白

写注册、付费、下载、版权、隐私、学习成本等问题。

## 适合谁

列出具体人群。

## 不适合谁

列出不推荐使用的人群。

## 怎么选

告诉用户什么时候选它，什么时候选替代工具。

## 新手步骤

给 3 到 5 步能照做的流程。

## FAQ

列出 3 到 5 个常见问题。

## 风险提醒

写清楚具体风险。

## 最后建议

给明确选择建议。
```

## 3. 文章模板

适用于 `/articles` 系统。

### 标题

不要百科式标题，要用体验、判断、避坑标题。

模板：

```txt
我试了 5 个免费图片工具，最适合小白的不是功能最多那个
```

### summary

一句话说明读者能得到什么。

模板：

```txt
如果你只是想压缩、抠图、改尺寸，不一定需要复杂软件，先选能最快完成任务的工具。
```

### content

文章正文建议结构：

```md
## 先给结论

直接告诉用户怎么选。

## 我试了什么

说明测试范围，不要装成全网权威。

## 最大的坑是什么

指出小白最容易误判的地方。

## 具体怎么选

用场景拆解。

## 对比表

用表格帮助快速判断。

## 新手行动步骤

给 3 到 5 步。

## 风险提醒

说明版权、隐私、付费、下载来源等风险。

## FAQ

回答常见问题。

## 最后建议

给明确行动建议。
```

## 4. X 平台推广文案模板

平台：X。

风格可以更强，但不能造假，不能承诺不真实结果。

### 模板 A：避坑型

```txt
很多人找 AI 工具，只看“功能强不强”。

但小白真正会翻车的地方通常不是功能，而是：

1. 要不要注册
2. 免费版能不能导出
3. 会不会自动续费
4. 上传文件有没有隐私风险
5. 生成内容能不能商用

我把工具按“适合谁 / 哪里坑 / 怎么选”重新整理了一遍。

与其收藏 100 个工具，不如先学会判断哪个适合你。
```

### 模板 B：焦虑型，但不造假

```txt
现在最危险的不是不会用 AI。

而是你以为自己会用。

很多工具看起来免费，真正常用到导出、商用、高清、批量处理时才发现限制一堆。

我现在整理工具时只看 3 件事：

1. 小白能不能上手
2. 免费版到底够不够
3. 有没有版权、隐私、下载风险

工具不缺，判断能力才缺。
```

### 模板 C：工具推荐型

```txt
我不建议小白一上来就追“最强工具”。

最强通常意味着：

- 设置复杂
- 学习成本高
- 容易卡在部署
- 免费版限制多
- 出问题不知道怎么查

更现实的选择是：

先用一个能完成任务的工具，跑通流程，再考虑进阶替代品。
```
