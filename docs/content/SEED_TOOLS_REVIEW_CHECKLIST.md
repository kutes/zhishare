# 首批工具人工复核清单 v1 - 真实写手风 GEO 版

## 1. 复核目标

seed-tools-v1.csv 只是草稿，不允许直接导入生产数据库。

每条工具发布前必须人工复核。

## 2. 基础信息必查

每条工具必须检查：

- 工具是否真实存在
- 官网链接是否正确
- 是否仍在运营
- 免费 / 付费信息是否准确
- 是否需要账号
- 是否需要下载客户端
- 是否支持中文
- 是否需要英文
- 是否涉及上传私人数据
- 是否涉及版权风险
- 是否适合中国大陆用户访问
- 是否有更合适的替代品
- 是否有明显不适合小白的门槛
- 是否有官方文档或帮助中心
- 是否有商用授权注意事项

## 3. 真实体验必查

发布前必须补充：

- 我实际用它做了什么？
- 哪一步最顺？
- 哪一步最卡？
- 有没有明显翻车点？
- 小白会不会误解？
- 免费版到底够不够用？
- 我会不会推荐给朋友？
- 我不推荐给哪类人？
- 有没有更适合小白的替代品？

## 4. 禁止直接发布的情况

出现以下情况不能发布：

- 官网无法确认
- 价格无法确认但文案写死
- 功能无法确认但文案写死
- 没有实测却写“我试过”
- 没有踩坑却写“我踩坑”
- 需要侵权下载
- 需要破解
- 有灰产属性
- 夸大收益
- 冒充官方
- 涉及敏感个人数据但没有风险提醒
- 涉及商用素材但没有授权提醒

## 5. 推荐发布顺序

第一批优先复核：

1. ChatGPT
2. Cursor
3. Figma
4. Canva
5. Notion
6. Obsidian
7. remove.bg
8. TinyPNG
9. GitHub
10. Vercel

原因：

- 用户认知高
- 搜索意图明确
- 适合做内链
- 适合关联新手教程
- 适合首页精选

## 6. 复核后状态

复核后更新 status：

- draft：草稿
- review：正在复核
- ready：可导入
- published：已发布
- archived：暂不使用

## 7. 后续导入规则

只有 status 为 ready 的工具才允许进入导入脚本。

禁止把 draft 直接导入生产数据库。
