## 3. 状态定义

### draft

只有草稿，还不能进入发布判断。

### review

已经有草稿和观察记录，但仍需要人工复核。

### ready

可以进入后台录入或发布前排版。

### published

已经发布到站点。

当前这两篇文章都只能保持 review。

## 4. 推荐等级定义

### 推荐

已经完成充分复测，风险边界清楚，可以作为推荐内容发布。

### 谨慎推荐

完成主要复测，但还存在明确限制，需要带风险提示发布。

### 暂不推荐

方向值得关注，但还缺关键复核，不适合写成推荐。

### 不建议使用

风险明显，或者官方来源 / 安全边界无法确认。

当前这两篇文章都只能保持“暂不推荐”。

## 5. 发布底线

禁止把以下内容直接发布为推荐：

- 没有稳定输出的隐私工具；
- 没有安全审计的 secret 分享工具；
- 没有官方来源核验的工具；
- 没有真实观察记录的工具；
- 不适合人群不明确的工具；
- 风险声明不清楚的工具；
- 只靠 ahhhhfs 线索的工具。
