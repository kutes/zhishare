### description

TinyPNG 是一个图片压缩工具。它解决的问题很单一：在尽量保持视觉效果的情况下，把图片文件变小。

很多新手做网站或写文章时，会直接上传原图。一张图片可能几 MB，页面打开就会变慢。这个时候，不一定需要复杂修图软件，先用 TinyPNG 压一下，往往就能明显改善加载速度。

但要注意，压缩不是魔法。图片体积变小，多少会有信息损失，只是好的压缩工具能让这种损失不那么明显。所以我建议永远保留原图，压缩图只作为发布版本使用。

### pros

使用简单，拖入图片就能压缩
适合网站和文章图片优化
对小白很友好，不需要懂复杂参数
支持常见图片格式
可以作为发布前的固定检查步骤

### cons

它不是修图工具，不能改善构图和清晰度
压缩过度可能影响画质
大批量或高级能力可能有付费限制
不要把压缩后的图片当唯一原图保存
不适合处理需要无损保留细节的专业图片

### risk_notice

TinyPNG 适合压缩发布用图片，但不要覆盖原图。尤其是商品图、客户图、设计源文件导出的图片，应该先备份原图，再生成压缩版。

如果图片本身模糊，压缩不会让它变清楚。压缩只能减少体积，不能替你修图。
