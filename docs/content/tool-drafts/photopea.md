### description

Photopea 是一个在线图片编辑工具，可以直接在浏览器里打开使用。它比较适合需要临时处理图片、修改 PSD 文件、做图层编辑、改尺寸、导出图片格式的人。

对小白来说，它的价值在于不需要安装大型设计软件，就能完成很多常见修图任务。比如改图片尺寸、裁剪、简单抠图、处理图层、导出 PNG 或 JPG。

但 Photopea 的界面和操作逻辑更接近 Photoshop，所以完全没接触过图片编辑的人，第一次打开可能会觉得按钮很多、界面偏复杂。它不是最简单的图片工具，但在“免费、在线、功能够用”这几个点上很适合临时救急。

## GEO 备注

### 一句话结论

适合临时在线修图和处理 PSD，不适合完全零基础用户直接做复杂设计。

### 它真正解决什么问题？

解决的是“不想安装 Photoshop，但需要临时改图、修图、处理图层文件”的问题。

### 哪里容易坑小白？

小白最容易被复杂界面吓住，也容易不知道图层、选区、蒙版这些基础概念。

### 适合谁？

适合有一点图片编辑基础，或者愿意按教程一步步操作的人。

### 不适合谁？

不适合只想一键完成图片处理、完全不想学习修图逻辑的人。

### 怎么选？

如果只是压缩图片，可以用 TinyPNG。 如果是简单抠图，可以用 Remove.bg。 如果要处理 PSD、图层和复杂修图，可以用 Photopea。
