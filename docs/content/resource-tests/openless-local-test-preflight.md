# OpenLess 本机实测可行性预检 v1

Status: preflight

## 1. 当前目标

本步骤只判断当前机器是否适合进入 OpenLess 本机实测。
本步骤不安装 OpenLess，不运行 OpenLess，不配置真实凭证，不录制真实语音。

## 2. 本机环境

记录：

- 操作系统
- 系统架构
- 是否 Windows
- 是否有音频设备
- 是否有 Chrome / Edge
- 是否有基础文本编辑器
- 是否适合做低敏测试环境

## 3. 已确认的官方边界

根据已落盘的官方来源备注，记录：

- OpenLess 是开源、本地优先的语音输入工具。
- 官网和 README 都说明音频、转写、词典、历史存在本地。
- 官网提到 Keychain 存凭证，`~/.openless` 存配置。
- 官网提到 BYO 凭证。
- 官网提到外部 ASR / LLM 边界。
- 官网提到麦克风和辅助功能权限。
- README 说明插入失败时会复制到剪贴板。

注意：

官方声明，不是本机实测结论。

## 4. 后续实测所需条件

后续如果进入真测，需满足：

- 用户明确同意下载并安装 OpenLess。
- 只使用虚构的样本。
- 不输入真实客户资料。
- 不输入真实公司文档。
- 不输入真实会议录音。
- 不输入真实 API Key。
- 不配置生产凭证。
- 记录麦克风权限。
- 记录辅助功能权限。
- 记录是否出现外部 ASR / LLM 请求。
- 记录输出是否进入当前光标。
- 记录插入失败时是否复制到剪贴板。

## 5. 当前不允许做的事

当前不允许：

- 下载 release
- 安装应用
- 运行应用
- 输入真实语音
- 配置生产凭证
- 把 observation sheet 改成 observed
- 写文章
- 发布文章

## 6. 当前建议

当前只完成安装前可行性预检。
下一步是否进入实测准备，需要用户明确同意。

## 7. 下一步建议

建议下一步只做安装前人工确认。
如果用户同意，再进入真正的安装前准备。
